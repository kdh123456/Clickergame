# 나의 바텐더 생활

자신만의 바를 운영 해보세요!!!

다양한 <b>칵테일</b>의 <b>숙련도</b>를 늘려 <b>자동화</b>를 사용하세요!

더 많이 팔아 <b>골드</b>를 얻어 재료를 뽑으세요

<b>조작 방법</b> :
  - 좌우이동 : AD
  - 점프 : SPACE
  - 아이템 먹기 : F
  - 능력 사용하기 : E
  - 일시 정지 : ESC
  
  ※ <b>주의!</b> 자동 세이브 안됨
